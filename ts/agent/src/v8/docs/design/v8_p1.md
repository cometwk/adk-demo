## Data Flow

```text
User Question → Agent (LLM) → Tool Call → Runtime → Store → Result
                                              ↓
                                   workspace.bindings (FactBinding[])
                                              ↓
                                   workspace.candidates (GlobalId[])
                                              ↓
                              $workspace.candidates (动态引用代换)
                                              ↓
                              parseGlobalId('Merch:M001') → 'M001'
```

## Key Design Decisions

1. **查询分工**: `graph_query` 做 traversal，`compute_query` 做 OLAP 聚合
2. **动态引用**: `compute_query` 支持 `$workspace.candidates` 引用，自动 `parseGlobalId` 解码
3. **Global ID**: `Merch:M001` 格式，`parseGlobalId()` 解析为 `{type, rawId}`
4. **FactStore 自动注入**: Runtime 自动注入低阶快照，Agent 通过 `bind_fact` 绑定高阶语义断言
5. **Zod v4 Strict**: 所有 Query Schema 使用 `.strict()` 拒绝未知字段
