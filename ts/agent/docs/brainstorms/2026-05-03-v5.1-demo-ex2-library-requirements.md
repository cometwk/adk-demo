# demo/ex2：图书馆借书决策场景

## Problem Frame

复用 v5.1 框架层，创建新的图书馆借书决策场景。Agent 需判断读者能否借书，生成候选方案、收集证据、给出推荐。

## Requirements

**场景本体**
- R1. Reader 类型：包含 `borrowedCount`（已借数量）、`hasOverdue`（是否有逾期）
- R2. Book 类型：包含 `daysOnShelf`（上架天数）、`title`（书名）
- R3. 关系：`wants_to_borrow`（Reader → Book）

**场景约束**
- R4. `borrow_limit_3`：读者最多借 3 本书（hard_constraint）
- R5. `new_book_restriction`：上架不满 7 天的书不能外借（hard_constraint）
- R6. `overdue_ban`：有逾期记录的读者不能借新书（hard_constraint）

**场景数据**
- R7. 读者小明：已借 1 本书，无逾期
- R8. 书《三体》：上架 10 天
- R9. Agent 目标："判断小明能否借《三体》"

**文件结构**
- R10. 创建 `src/v5.1/demo/ex2/` 目录，包含 main.ts, schema.ts, constraints.ts, seed.ts

## Success Criteria
- 运行 `npm run v5.1:ex2` 输出正确的决策结论
- Agent 收集证据、应用规则、给出推荐
